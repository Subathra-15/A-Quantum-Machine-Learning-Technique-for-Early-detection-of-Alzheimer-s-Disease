# A Quantum Machine Learning Technique for Early Detection of Alzheimer's Disease

## Project Title
A Quantum Machine Learning Technique for Early Detection of Alzheimer's Disease using Handwriting Analysis

## Author
Subathra-15

## Date
2026

## Description
This project uses Quantum Machine Learning to detect Alzheimer's Disease at an early stage by analyzing handwriting samples from the DARWIN dataset.

## Dataset
DARWIN (Diagnosis Alzheimer with HaNdwriting) dataset from UCI Machine Learning Repository
- 89 Alzheimer's patients
- 85 healthy controls
- 25 handwriting tasks
- 18 features per tasks

## How to Run
1. Install Python 3.8 or higher
2. Install required libraries: qiskit, scikit-learn
3. Download DARWIN dataset 
4. Open Jupyter notebooks
5. Run cells in order

## Requirements
- Python 3.8+
- Qiskit
- scikit-learn
- numpy
