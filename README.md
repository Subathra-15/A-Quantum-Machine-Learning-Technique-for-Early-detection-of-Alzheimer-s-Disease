# A Quantum Machine Learning Technique for Early Detection of Alzheimer's Disease

## Project Title
A Quantum Machine Learning Technique for Early Detection of Alzheimer's Disease using Handwriting Analysis

## Author
Subathra-15

## Date
2026

## Description
This project uses Quantum Machine Learning to detect Alzheimer's Disease at an early stage by analyzing handwriting samples from the DARWIN dataset.

## Dataset
DARWIN (Diagnosis Alzheimer with HaNdwriting) dataset from UCI Machine Learning Repository
- 89 Alzheimer's patients
- 85 healthy controls
- 25 handwriting tasks
- 18 features per task

## Files in this Repository
1. Classical (table 2).ipynb - Classical SVC results
2. Classical (table 3 - all 25 and 5).ipynb - Majority voting results
3. Fig2.ipynb - SVM visualization
4. Fig4.ipynb - Eigenvalue curves
5. Fig5.ipynb - Condition number analysis
6. Fig6.ipynb - Entropy analysis
7. Quantum(table1 - 6 qubit).ipynb - 6-qubit quantum classifier
8. Quantum(table1 - 8 qubit).ipynb - 8-qubit quantum classifier
9. Quantum(table1 - 12 qubit).ipynb - 12-qubit quantum classifier
10. Quantum(table2 - 12 qubit - graphic).ipynb - Feature subset analysis
11. Table3-9qubit.ipynb - 9-qubit majority voting
12. Table4-Fig7.ipynb - Noise analysis


## How to Run
1. Install Python 3.8 or higher
2. Install required libraries: qiskit, scikit-learn
3. Download DARWIN dataset 
4. Open Jupyter notebooks
5. Run cells in order
